# BMAD × OpenClaw (Skill Package)

This repository packages the BMAD Method for use inside OpenClaw.

## What’s inside
- **SKILL.md** — how to use BMAD within OpenClaw
- **scripts/bmad-launcher.sh** — helper to generate `sessions_spawn` calls
- **references/BMAD_Openclaw/** — full BMAD prompts, templates, checklists, and workflow files

## Intended use
Use this as an OpenClaw skill to run structured multi‑agent software delivery:
PRD → Architecture → Epics → Stories → Dev → Review → QA → Retro.
